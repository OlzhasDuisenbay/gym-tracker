# Gym Tracker

## Description
Gym Tracker is a full-stack web application built with Angular and Django REST Framework.
It allows users to track workouts, exercises, and progress.

## Features
- User authentication
- Create, update, and delete workouts
- Add exercises
- Track progress
- API integration
